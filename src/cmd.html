cmd¶

Misc¶

  1. 条件命令

    命令1 & 命令2 & 命令3 : 无论前面命令是否故障，照样执行后面

    命令1 && 命令2 && 命令3 : 仅当前面命令成功时，才执行后面

    命令1 || 命令2 || 命令3 : 仅当前面命令失败时，才执行后面

  2. @ : 关闭紧跟其后的一条命令的回显

    echo "some text" : 回显文本 “some text”

    echo off : 关闭其他（不包括这条命令自身）命令的回显

    @echo off : 关闭其他及这条命令的回显

  3. 注释： ::

  4. set [变量名] # 查看环境变量的值，无参数时查看所有环境变量的值

  5. set <变量名>=[新值] # 修改环境变量的值，新值可以留空；追加内容 set <变量名>=%变量名%;变量内容

  6. 切换代码页：chcp <code page>

    常用的 code page:

    • 950 : 繁体中文

    • 65001 : UTF-8 代码页

    • 936 : 简体中文默认的 GBK

    • 437 : MS-DOS 美国英语

警告

所有的在 cmd 命令行下对环境变量的修改只对当前窗口有效，不是永久性的修改。也 就是说当关闭此 cmd 命令行窗口后，将不再起作用。
